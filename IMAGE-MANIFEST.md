# Image File Manifest — Maison Lumière

Every image slot on the site now expects a **specific filename** in `/public/images/`. Drop a correctly-named file in and it appears automatically — no code editing required. If a file is missing, the site shows a minimal "photo needed" placeholder instead of breaking.

Recommended: JPG or WebP, sRGB, at least 2000px on the long edge for full-width shots, 1200px for cards/thumbnails.
